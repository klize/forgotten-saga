# Forgotten Saga (FSAGA) Mod

포가튼사가(Forgotten Saga) 확장 모드입니다.

## Download

최신 버전은 [다운로드 페이지](https://klize.github.io/forgotten-saga/)에서 받으실 수 있습니다.

또는 [Releases](https://github.com/klize/forgotten-saga/releases) 페이지에서 직접 다운로드하세요.

## 설치 방법

1. ZIP 파일을 다운로드합니다.
2. 게임 폴더에 압축을 풀어 덮어씌웁니다.
3. `FSAGA_Loader.exe`를 실행합니다.
