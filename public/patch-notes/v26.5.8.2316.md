# v26.5.8.2316
**2026-05-08**


### 1. 아이겐필드 빙결 시스템 신규 추가
* 아이겐필드 마법에 **빙결 효과**가 추가되었습니다. 적중한 적은 얼어붙어 일정 턴 동안 행동하지 못합니다.
* 빙결 지속 턴수는 대상의 저항치에 따라 달라집니다.
* 빙결 해제 시 데미지가 들어갑니다.
* 빙결된 유닛은 아군/적군 모두 타겟 선택에서 제외됩니다.
* **부사**, **블러드스톤**은 빙결 면역입니다.
* 전투 종료 시 빙결 상태가 일괄 정리됩니다.
* 스탯 오버레이에 빙결 상태가 표시됩니다 (독/출혈도 함께 표시).

### 2. 석화 시스템 신규 추가
* **바실리스크**의 석화 공격이 구현되었습니다. 일정 확률로 적을 석화시킵니다.
* 석화된 유닛은 모션이 정지되고 턴이 스킵됩니다 (CLUT 색상 변경 적용).
* 석화된 유닛에 물리 공격이 들어가면 깨져서 사망합니다. 깨질 때 폭발 연기 이펙트가 재생됩니다.
* **안티스톤드** 마법이 추가되었습니다 — 석화를 해제합니다.
* **석화치료약**이 상점에 등록되었습니다.
* 반지/팔찌의 저항 효과로 석화에 저항할 수 있습니다.
* 석화 피격 모션 / 반격 / 저항 아이템 훅이 모두 정리되었습니다.

### 3. 난이도 설정 시스템 추가 (difficulty_config v1.0.0)
* `DIFFICULTY.txt` 파일로 게임 난이도를 세밀하게 조정할 수 있습니다.
* 11개 섹션으로 구성 — 몬스터 스탯 / 드랍 확률 / 경험치 / 골드 등이 있으나 석화 부분만 테스트 되었습니다. 나머지는 모두 테스트되지 않은 부분입니다.
* 첨부된 `DIFFICULTY.txt` 를 확인 후 게임 폴더에 위치시켜주세요.

### 4. 마법 리메이크
* **쟈이거스피라**: 단일 타겟 → **광역 공격**으로 전환 (AoE 5). 타겟별 동시 이펙트로 연출됩니다.
* **라이트닝포스**: 미사일 순차 발사 + 거리 기반 속도 + 방향별 모션 전환 + 사운드 보강.
* **아이겐스톰**: MP 소모 35 → **30** 으로 하향.

### 5. 반지 쉴드링 마법 피해 감소
* 반지 라이트닝/파이어링/아이스 쉴드링 효과가 누락된 것으로 보여 패치했습니다.
이즈렌님 방송에서도 베리어링 효과 버그를 착각했었는데, 이들도 혹시 잘못 패치되었다면 이부분은 알려주세요.

### 6. CLUT 프리뷰 확장
* 마법 / 어빌리티 타겟 선택 시 광역 범위 / 관통 경로가 색상으로 표시됩니다.
* **광역 마법** 프리뷰 적용.
* **단일 타겟 마법** 프리뷰 적용.
* **창 관통 공격** 프리뷰 적용.
* **어빌리티 / 물리 공격** 프리뷰 적용.
* 디버프 마법은 빨간색으로 구분 표시됩니다.

### 7. motion_critical 외형 기반 매칭으로 전환
* 크리티컬 모션 매칭이 **캐릭터 이름이 아닌 외형(body SPB) 기반**으로 동작합니다.
* 같은 외형을 쓰는 다른 캐릭터에도 자동으로 적용됩니다.
* `MOTION_CRITICAL.txt` 의 키 형식이 바뀔 수 있으니 첨부 파일을 확인해주세요.
